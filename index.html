What is JavaScript?

It was...

... originally developed by Netscape as a means to add dynamic and interactive elements to websites. The father of JavaScript is Brendan Eich. And by the way Brendan created JavaScript in 10 Days. Bravo to that!

It is a...

... scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else. Wow, it is amazing what you can achieve with a few lines of JavaScript code. Try it out below!

And ...

... alongside HTML and CSS JavaScript is one of the core technologies of the World Wide Web. You can say it's the third layer of the "standard web technologies cake". Sounds like a really nice cake, don't you think?
